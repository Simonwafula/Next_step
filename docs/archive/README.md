# Archive

This directory holds historical or workflow-specific documents that are kept for
reference but are no longer intended to live at the repo root.

## Root Docs Archive

- `root/CHANGES.md`
- `root/IMPLEMENTATION_SUMMARY.md`
- `root/PR_DESCRIPTION.md`
- `root/SCOPE_CLEANUP.md`
- `root/TEST_RESULTS.md`
- `root/agent-work.md`

Keep active, user-facing, or operationally current documents at the repo root or
under `docs/`. Move one-off summaries, PR scratch files, and historical reports
here when they are no longer part of the main workflow.

# User Journey

## 1) Visitor / Guest
- Lands on the homepage and learns what the app does.
- Enters skills, courses, or a full sentence in the search bar.
- Reviews matched roles and opens verified job links.
- Sees a prompt to create an account for saved roles and tailored results.

## 2) Registered Job Seeker
- Signs up with email/password or Google.
- Adds location and top skills during signup (or later in profile).
- Runs searches and receives more relevant recommendations.
- Saves roles, returns to the account view to track saved roles.
- Requests password resets or updates profile data as needed.

## 3) Returning User
- Signs in to see profile snapshot and saved roles.
- Refines searches with location and seniority.
- Applies for roles and keeps saved roles list current.

## 4) Admin / Operator
- Curates source lists and ensures coverage quality.
- Runs ingestion jobs or schedules monitoring.
- Reviews data freshness and resolves ingestion errors.
- Updates feature flags and environment configuration.
